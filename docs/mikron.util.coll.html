mikron.util.coll

Fast collection operations.

count

(count coll)

Returns the length of a vector coll.

every?

(every? pred coll)

Returns true if pred returns true for each element of a vector coll, false otherwise.

into!

macro

(into! coll transient? n expr)

Repeatedly evaluates expr n times, collecting the results into a collection coll. Uses transient operations if transient? is true.

into-kv!

macro

(into-kv! coll transient? n key-expr value-expr)

Repeatedly evaluates key-expr and value-expr n times, collecting the results into a map coll. Uses transient operations if transient? is true.

nth

(nth coll index)

Returns the value of a vector coll at the position index.

rand-nth

(rand-nth coll)

Returns a random value from a vector coll.
