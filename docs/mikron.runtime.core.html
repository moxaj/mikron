mikron.runtime.core

Core namespace. Contains the public API.

allocate-buffer

(allocate-buffer size)

Allocates a new buffer with the given size.

defschema

macro

(defschema & args)

Globally registers a reified schema for the given schema definition, with the given name.

diff

(diff schema value-1 value-2)

Returns the diff between the old (value-1) and the new (value-2) value, both conforming to schema. Wraps the return value with DiffedValue for pack and undiff consumption.

diff*

(diff* schema value-1 value-2)

Returns the diff between the old (value-1) and the new (value-2) value, both conforming to schema.

gen

(gen schema)

Generates a new value which conforms to schema.

interp

(interp schema value-1 value-2 time-1 time-2 time)

Calculates a new value of an entity at time, given two other values (value-1 and value-2, both conforming to schema) and their respective timestamps (time-1 and time-2). Uses linear interpolation.

pack

(pack schema value)

Packs value, which must conform to schema, and may be an instance of DiffedValue.

register-schema!

(register-schema! schema-name schema)

Registers a reified schema with the given name.

resolve-schema

(resolve-schema arg)

Returns a resolved schema for the given argument.

schema

macro

(schema & args)

Returns a reified schema for the given schema definition.

schema*

(schema* & args)

Given a schema definition, returns the unevaluated code to produce a reified schema.

schema?

(schema? arg)

Returns true if arg is an instance of Schema, false otherwise.

set-byte-buffer-factory!

(set-byte-buffer-factory! factory)

Sets the byte buffer factory.

undiff

(undiff schema value-1 value-2)

Returns the original value from the old (value-1) and the diffed (value-2) value. The old value must conform to schema. value-2 must be an instance of DiffedValue.

undiff*

(undiff* schema value-1 value-2)

Returns the original value from the old (value-1) and the diffed (value-2) value. The old value must conform to schema.

unpack

(unpack schema binary)

Unpacks a value (which conforms to schema) from the binary value binary.

valid?

(valid? schema value)

Returns true if value conforms to schema, false otherwise.

with-buffer

macro

(with-buffer buffer & body)

Executes all the expressions of body in the context of buffer.
