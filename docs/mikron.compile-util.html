(find-by f form)

Walks form and collects all values for which the predicate f returns true. Filter duplicates.

(with-evaluated syms & body)

Executes each expression of body in the context of each symbol in syms bound to an evaluated value. Can be used to prevent accidental multiple evaluation in macros.
